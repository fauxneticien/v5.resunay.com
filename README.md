# v5.resunay.com

Personal website for **Nay San** — version 5. 🚧 Rebuild in progress.

Hand-written static site, built locally and deployed to [Cloudflare Pages](https://pages.cloudflare.com/).

## Develop

```sh
docker compose up -d   # start the dev container
task preview           # serve ./public at http://localhost:8919
task build             # build into ./public
task push              # deploy to Cloudflare Pages
```
